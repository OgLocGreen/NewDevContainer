## Allgemeine Aktionen
- **Befehlspalette öffnen**: `Strg + Umschalt + P`
- **Einstellungen öffnen**: `Strg + ,`
- **Neues Fenster öffnen**: `Strg + Umschalt + N`
- **Aktives Fenster schließen**: `Strg + W`

## Dateien und Navigation
- **Datei öffnen**: `Strg + O`
- **Datei speichern**: `Strg + S`
- **Alle Dateien speichern**: `Strg + Alt + S`
- **Zu Datei springen**: `Strg + P`
- **Zwischen Dateien wechseln**: `Strg + Tab`
- **Seitenleiste ein-/ausblenden**: `Strg + B`
- **Datei schließen**: `Strg + W`

## Code bearbeiten
- **Ausschneiden**: `Strg + X`
- **Kopieren**: `Strg + C`
- **Einfügen**: `Strg + V`
- **Rückgängig**: `Strg + Z`
- **Wiederherstellen**: `Strg + Umschalt + Z`
- **Einrücken**: `Tab`
- **Ausrücken**: `Umschalt + Tab`
- **Zeile löschen**: `Strg + Umschalt + K`
- **Zeile kopieren**: `Alt + Umschalt + Pfeil nach unten/oben`
- **Auskommentieren (Toggle):** `Strg + Shift + /`
- **Auskommentieren**: `Strg + K, Strg + C`
- **Einkommentieren**: `Strg + K, Strg + U`

## Suchen und Ersetzen
- **Suchen**: `Strg + F`
- **Ersetzen**: `Strg + H`
- **Nächster Treffer**: `F3`
- **Vorheriger Treffer**: `Umschalt + F3`

## Mehrere Cursor
- **Cursor hinzufügen (oben/unten)**: `Alt + Umschalt + Pfeil nach oben/unten`
- **Wort auswählen**: `Strg + D`
- **Alle Vorkommen eines Wortes auswählen**: `Strg + Umschalt + L`

## Terminal
- **Neues Terminal öffnen**: `Strg + ö`
- **Terminal ein-/ausblenden**: `Strg + ^`
- **Terminal schließen**: `Strg + Umschalt + ö`

## Debugging
- **Debugging starten**: `F5`
- **Debugging beenden**: `Umschalt + F5`
- **Zum nächsten Haltepunkt springen**: `F9`

## Git (Versionskontrolle)
- **Git-Ansicht öffnen**: `Strg + Umschalt + G`
- **Änderungen stagen**: `Strg + K, Strg + U`
- **Commit ausführen**: `Strg + Enter`
- **Git-Logs anzeigen**: `Strg + Umschalt + H`

## Split-Screen
- **Editor aufteilen (Vertikal)**: `Strg + \`
- **Zwischen Gruppen wechseln**: `Strg + K, Strg + Pfeiltasten`

## Nützliche Shortcuts
- **Autocomplete anzeigen**: `Strg + Leertaste`
- **Code formatieren**: `Umschalt + Alt + F`
- **Gehe zu Definition**: `F12`
- **Vorschau der Definition anzeigen**: `Alt + F12`
- **Zu Referenzen springen**: `Umschalt + F12`
- **Fehler anzeigen**: `Strg + Umschalt + M`

---
> Hinweis: Die Shortcuts können unter **Datei → Einstellungen → Tastenkombinationen** angepasst werden.
